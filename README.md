Mobile Webshop Project\
Made by: Bence Benko

Description:
A web system which allows the user to browser through different mobile phones, filter them\
by price, color, brand or storage size. They can register an account, put items in their cart\
and place an order. The homepage has a NEW section, a SALE section and a OUTGOING section.\
The admin can add new products, update or reset information. Each item has their own information site.

**Tech Stack:**\
Frontend: Angular\
Backend: Django\
Database: MySQL

**Features:** 
- Basic CRUD operations with a web GUI by only authenticated users (admin)
- Session handling
- Registration for new users
- REST API with HTTP requests
